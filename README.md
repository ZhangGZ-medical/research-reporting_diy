# research-reporting_diy

通用结构化调研报告生成技能，适用于所有调研任务。

## 功能

按六阶段工作流执行完整调研任务，从任务澄清到报告交付全程自动化：
1. **内容澄清** — 50轮Q&A上限，聚焦内容非格式
2. **技能分析** — 识别所需技能，缺失则安装
3. **提示词生成** — 含八要素的完整执行提示词
4. **用户确认** — 展示提示词，等待"确认执行"
5. **执行调研** — 文件读取 + Agent并行调研 + Web搜索
6. **报告撰写与交付** — MD + DOCX双格式，国自然基金引文规范

## 触发词

先给出良好的提示词、学术引用格式、深度调研、科技文献引用格式、多维度分析、生成调研报告、帮我调研

## 核心原则

| 原则 | 规则 |
|------|------|
| 交付格式 | MD + DOCX（纵向A4） |
| 引用格式 | 国自然基金引文格式 |
| 澄清聚焦 | 只问内容，不问格式 |
| 用户确认 | 提示词生成后必须等待确认 |
